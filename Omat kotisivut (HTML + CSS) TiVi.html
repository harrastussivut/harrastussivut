Omien kotisivujesi julkaisuohjeet GitHub-pages -palvelussa:

Vaihe 1: Luo GitHub-repositorio:

  1. Mene osoitteeseen https://github.com/ ja kirjaudu sisään tai luo uusi tili, jos sinulla ei vielä ole sellaista.
  2. Klikkaa oikeasta yläkulmasta "+"-merkkiä ja valitse "New Repository" (Uusi repositorio).
  3. Anna repositoriolle nimi, esimerkiksi "harrastus-sivu".
  4. Voit jättää kuvauksen tyhjäksi tai lisätä haluamasi kuvauksen.
  5. Valitse asetuksista "Public" (Julkinen), jos haluat sivun olevan julkisesti saatavilla.
  6. Voit valita lisäasetuksia tarpeesi mukaan.
  7. Klikkaa lopuksi "Create repository" (Luo repositorio).

Vaihe 2: Lisää tiedostot repositoryyn:

  1. Kun repositorio on luotu, avaa se.
  2. Klikkaa "Add file" (Lisää tiedosto) ja valitse "Upload files" (Lataa tiedostot).
  3. Lataa sekä index.html että styles.css -tiedostot.
  4. Klikkaa "Commit changes" (Vahvista muutokset) tiedostojen lisäämisen jälkeen.

Vaihe 3: Julkaise GitHub Pages:

  1. Mene takaisin repositorion pääsivulle.
  2. Klikkaa "Settings" (Asetukset) -välilehteä.
  3. Selaa alaspäin kohtaan "Pages".
  4. Klikkaa "None" -valikkoa ja valitse "main" (tai muu päähaarasi nimi) -haara.
  5. Valitse "root" (juurihakemisto) -kansiosta, jotta GitHub Pages käyttää pääasiallista hakemistoa sivujesi julkaisuun.
  6. Klikkaa "Save" (Tallenna).

Vaihe 4: Odota julkaisua:

GitHub Pages vie hetken prosessoida sivujasi ja julkaista ne. Kun julkaisu on valmis, saat ilmoituksen. Voit avata julkaistun sivun selaimessasi osoitteessa https://käyttäjänimi.github.io/reponimi, jossa käyttäjänimi on GitHub-käyttäjäsi nimi ja reponimi on repositorion nimi.

Huomioi, että jos käytössäsi on jokin muu päähaara kuin main, korvaa se ohjeissa mainitulla haaran nimellä.

Nyt sivusi pitäisi olla julkaistuna GitHub Pages -palvelussa ja saatavilla osoitteessa https://käyttäjänimi.github.io/reponimi.

Omien kotisivujesi julkaisuohjeet Pieksämäki!!:

Sivustosi voit siirtää palvelimelle "pmk.esedu.fi" esim. Bitvise SSH Clientin avulla. :

  1. Lataa se netistä
  2. Asenna
  3. Käynnistä
  4. Pyydä opettajasi tai taitovalmentajasi tekemään tunnukset pmk.esedu.fi-palvelimelle.
  5. Server: Host: pmk.esedu.fi - Port: 22
  6. Authentication: Username: etunimi.sukunimi (ei ääkkösiä!)
  7. Paina: ”Log in” -nappulaa (vasen alareuna)
  8. Salasana: Opettajan antama salasana
  9. Ensimmäisen kirjautumisen yhteydessä salasana kannattaa vaihtaa seuraavasti:
    • valitse kirjautumisen jälkeen avautuva konsoli-ikkuna. Jos konsoli ei avaudu, voit painaa New terminal console- painiketta.
    • kirjoita konsoliin komento: passwd
    • tämän jälkeen noudata ohjeita: kirjoita ensi nykyinen salasanasi ja sitten uusi salasana kahteen kertaan. Älä hämmästy, vaikka et näe kirjottamaasi salasanaa tai edes tähtimerkkejä. Linux ei näytä näytöllä edes salasanan merkkien määrää.
    • nyt salasanasi on vaihdettu.

Sitten siirrät Bitvise SFTP-ikkunan kautta tavarat omalta koneeltasi (vasen ikkuna) omaan kansioon "etunimi.sukunimi" (oikea ikkuna). Oikea kansio pitäisi näkyä suoraan ikkunan oikealla puolella.

HUOM! Jos et työskentele koulun verkossa, joudut muodostamaan ensin VPN-yhteyden.
